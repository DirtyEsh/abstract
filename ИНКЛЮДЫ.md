Короткая **сводка `#include` — что для чего** (только из того, что выше).

---
```
#include <cstdlib>     // для std::abort()
#include <utility>   // для std::pair
```
## Итераторы и контейнеры

```cpp
#include <vector>
#include <string>
```

- `std::vector`
    
- `begin() / end() / rbegin() / rend()`
    
- `iterator`, `const_iterator`
    

---

## Алгоритмы STL

```cpp
#include <algorithm>
```

- `std::fill`
    
- `std::copy`, `std::copy_n`
    
- `std::generate`, `std::generate_n`
    
- `std::transform`
    
- `std::sort`
    
- `std::unique`
    
- `std::remove`, `std::remove_if`
    
- `std::reverse`
    
- `std::distance`
    
- `std::advance`, `std::next`, `std::prev`
    

---

## Лямбды

```cpp
// ничего
```

- лямбды — часть языка
    
- `[]`, `mutable`, захваты — без include
    

---

## Итераторные адаптеры

```cpp
#include <iterator>
```

- `std::back_inserter`
    
- `std::distance`
    
- `std::advance`
    
- `std::next`
    
- `std::prev`
    

(часто тянется через `<algorithm>`, но **правильно** — `<iterator>`)

---

## Математика / тригонометрия

```cpp
#include <cmath>
#include <numbers>
```

- `sin`, `cos`
    
- `std::numbers::pi`
    

---

## Типы фиксированной ширины

```cpp
#include <cstdint>
```

- `int16_t`, `uint32_t`, etc.
    

---

## Ввод / вывод (для примеров)

```cpp
#include <iostream>
```

- `std::cout`, `std::endl`
    

---

## Мини-TL;DR (одним взглядом)

```
vector / string        → <vector> <string>
алгоритмы              → <algorithm>
advance / back_inserter→ <iterator>
лямбды                  → ничего
sin / cos / pi         → <cmath> <numbers>
int16_t                → <cstdint>
```

Если хочешь — следующим шагом можно сделать **таблицу “алгоритм → include”** или **что точно НЕ надо инклудить в `.h`**.
