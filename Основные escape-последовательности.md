



### `\n` - Новая строка (Line Feed)
```cpp
std::cout << "Первая строка\nВторая строка";
// Вывод:
// Первая строка
// Вторая строка
```

### `\t` - Табуляция (Horizontal Tab)
```cpp
std::cout << "Имя:\tАнна\nВозраст:\t25";
// Вывод:
// Имя:    Анна
// Возраст:    25
```

## Другие важные escape-последовательности:

| Последовательность | Значение | Описание |
|-------------------|----------|----------|
| `\\` | Обратная косая черта | Выводит сам символ `\` |
| `\"` | Двойная кавычка | Для вывода кавычек внутри строки |
| `\'` | Одинарная кавычка | Для символов char |
| `\?` | Вопросительный знак | Редко используется |
| `\0` | Нулевой символ | Конец строки в C-строках |
| `\a` | Звонок (Alert) | Звуковой сигнал (бипер) |
| `\b` | Backspace | Возврат на один символ |
| `\r` | Возврат каретки (Carriage Return) | Возврат в начало строки |
| `\f` | Form Feed | Прогон страницы (для принтеров) |
| `\v` | Вертикальная табуляция | Редко используется |
| `\ooo` | Восьмеричный код | Символ по восьмеричному коду |
| `\xhh` | Шестнадцатеричный код | Символ по hex-коду |

## Примеры использования:

```cpp
#include <iostream>
#include <string>

int main() {
    // Путь к файлу
    std::string path = "C:\\Users\\Name\\Documents\\file.txt";
    std::cout << path << std::endl;  // C:\Users\Name\Documents\file.txt
    
    // Цитата
    std::string quote = "Он сказал: \"Привет!\"";
    std::cout << quote << std::endl;  // Он сказал: "Привет!"
    
    // Форматированный вывод
    std::cout << "Список:\n\t1. Первый пункт\n\t2. Второй пункт\n\t3. Третий пункт\n";
    
    // Символ по коду
    char heart = '\x3';  // Символ сердечка в некоторых кодировках
    std::cout << "I " << heart << " C++" << std::endl;
    
    // Возврат каретки (полезно для прогресс-баров)
    std::cout << "Загрузка: 0%";
    std::cout << "\rЗагрузка: 50%";  // Перезаписывает строку
    std::cout << "\rЗагрузка: 100%\n";
    
    return 0;
}
```

## Важные замечания:

1. **`\n` vs `std::endl`**:
   ```cpp
   std::cout << "Текст\n";        // Просто новая строка
   std::cout << "Текст" << std::endl;  // Новая строка + сброс буфера
   ```

2. **Сырые строки (Raw strings) C++11**:
   ```cpp
   // Без escape-последовательностей
   std::string path = R"(C:\Users\Name\Documents\file.txt)";
   std::string json = R"({
       "name": "John",
       "age": 30
   })";
   ```

3. **`\r\n` для Windows**:
   ```cpp
   // Windows использует \r\n для новых строк в файлах
   std::string text = "Line 1\r\nLine 2\r\nLine 3";
   ```

Если вы действительно имели в виду `/p` — такого escape-символа в стандарте C++ нет. Возможно, вы видели его в других контекстах или языках программирования.
