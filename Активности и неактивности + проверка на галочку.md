**`setEnabled()`:**  - Активирование кнопок

```
cpp

// Делает активной (кликабельной)
ui->button->setEnabled(true);  

// Делает неактивной (серой, не кликается)
ui->button->setEnabled(false);

```
---
**Да, `QSignalBlocker`:**

```cpp
#include <QSignalBlocker>

{
    QSignalBlocker blocker(ui->widget); // блокируем сигналы
    ui->widget->setValue(50);           // сигналы НЕ испускаются
} // автоматическая разблокировка

// Или в одну строку:
QSignalBlocker(ui->widget).widget()->setValue(50);
```

## **Для предотвращения зацикливания:**

```cpp
void MainWindow::on_spinbox_valueChanged(int value) {
    // Блокируем второй виджет чтобы не вызвать его слот
    QSignalBlocker blocker(ui->slider);
    ui->slider->setValue(value);
    // Слот on_slider_valueChanged НЕ вызовется
}

void MainWindow::on_slider_valueChanged(int value) {
    QSignalBlocker blocker(ui->spinbox);
    ui->spinbox->setValue(value);
}
```

## **Старый способ (не рекомендуется):**
```cpp
ui->widget->blockSignals(true);
ui->widget->setValue(50);
ui->widget->blockSignals(false);
```

## **Старый способ (не рекомендуется):**

cpp

ui->widget->blockSignals(true);
ui->widget->setValue(50);
ui->widget->blockSignals(false);

**`QSignalBlocker` лучше — сам разблокирует при выходе из scope, даже если будет исключение.**
