# Инициализация `optional` и структур

## `std::optional<T>`

```cpp
std::optional<int> opt1;           // Пустой (без значения)
std::optional<int> opt2{};         // То же самое, пустой
std::optional<int> opt3 = 42;      // Значение 42
std::optional<int> opt4{42};       // Значение 42
std::optional<int> opt5 = {};      // Пустой
std::optional<int> opt6 = std::nullopt; // Пустой (явно)
```

## Структуры

```cpp
struct Point { int x, y; };

// Скобки {} - uniform initialization (C++11)
Point p1{10, 20};     // Прямая инициализация
Point p2 = {10, 20};  // Копирующая инициализация
Point p3{};           // x=0, y=0 (value initialization)
Point p4{.x=10, .y=20}; // Designated init (C++20)

// Круглые скобки () - вызов конструктора
Point p5(10, 20);     // Только если есть конструктор
```

## Коротко:
- **`{}`** — современный способ, предотвращает сужение типов
- **`()`** — вызов конструктора
- **`=`** — копирующая инициализация
